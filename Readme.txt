🏛️ Cabinet Juridique - Site vitrine

📌 Présentation du projet

Ce projet est un site vitrine fictif de cabinet juridique réalisé dans le cadre de ma formation en première CIEL (Cybersécurité, Informatique et réseaux, Électronique).

Il a été développé dans un objectif d’apprentissage du développement web front-end.

🎯 Objectifs du projet
Découvrir la création d’un site web en HTML / CSS
Apprendre à structurer un projet web simple
Mettre en pratique les bases du développement front-end
Comprendre l’organisation d’un site multi-pages


🛠️ Technologies utilisées
HTML5
CSS3
JavaScript (si présent dans le projet)

📁 Structure du projet
/
├── index.html
├── contact.html
├── style.css
├── script.js (si utilisé)
└── images/


🌐 Fonctionnalités
Page d’accueil avec présentation du cabinet
Navigation entre plusieurs pages
Section services / équipe
Page de contact
Interface simple et lisible


📱 Améliorations possibles
Rendre le site responsive (mobile / tablette)
Améliorer le design (UI/UX)
Ajouter des animations CSS
Améliorer le formulaire de contact
Hébergement du site via GitHub Pages


📚 Compétences développées
Structuration d’une page web en HTML
Mise en forme avec CSS
Organisation d’un projet simple
Découverte du développement web front-end

👤 Auteur

Projet réalisé par un élève en première CIEL, dans un cadre pédagogique.
